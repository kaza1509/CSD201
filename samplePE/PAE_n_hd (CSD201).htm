Hướng dẫn làm và nộp bài thi CSD201_PE sử dụng phần mềm PEA_Client_Version2_CSD

 

Các bước cần thực hiện khi làm bài thi:

 

1. Download tệp PEA_Client_Verson2_CSD.zip (tên có thể khác chút ít do thay đổi phiên bản). Giải nén và chạy tệp PEALogin.exe. Xuất hiện màn hình đăng nhập như sau:

Sinh viên nhập vào Test name do giám thị cung cấp. Nhập user name, password (chính là user name và password vào mạng của trường) rồi nhấn Login.

 

 

2. Đăng nhập thành công thì sẽ xuất hiện cửa sổ sau:

Tạo thư mục để chứa các tệp given, ví dụ  C:\CSD_given (1)

Click  vào nút “Download given materials”, sau đó chọn thư mục (1)  và nhấn Ok để lưu. Click vào nút “Show paper” để xem đề, sử dụng các given project theo hướng dẫn đề hoàn thành bài thi theo yêu cầu đề.

 Chú ý:

Không dùng package trong các Project bài làm.

Khi code không sử dụng tiếng Việt có dấu.

Bài nộp từng project: Nếu làm hết thì sẽ nộp 3 lần, lần 1 nộp chọn Question no = 1, rồi click vào nút Browse, chọn project Q1, lần 2 chọc Question No = 2, chọn project Q2, lần 3 chọc Question No = 3, chọn project Q3.

 

Chú ý: Không nộp Project chưa làm. Ví dụ nếu bạn không viết code trong tệp Graph.java thì không cần nộp project Q3.
